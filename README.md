## Installation

### Prerequisite
- Installation of nodejs

### Steps
1. Add pathfinder to your mix.exs
```elixir
def deps do
  [
    {:pathfinder, "~> 0.1.0"}
  ]
end
```
1. Run `mix deps.get`
1. Copy files
`cp ../deps/pathfinder/node ./`
1. Install NodeJS dependencies
`cd node && npm install`
